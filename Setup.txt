Setup:
mkdir server
cd server
npm init -y

cd..
npx create-react-app client
cd client

Install Packages (Frontend/client)
multer (store files)
- npm install multer

material UI ( styling and react packages) (Frontend/client)
- npm install @mui/material @emotion/react @emotionstyled axios react-router-dom
